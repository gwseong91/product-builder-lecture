당첨 등수 안내

로또 6/45는 1~45번 중 6개를 맞히는 게임입니다. 추첨 번호와 일치하는 개수에 따라 등수와 당첨금이 결정됩니다.

1등 6개 번호 모두 일치 확률 1/8,145,060
2등 5개 번호 + 보너스 번호 확률 1/1,357,510
3등 5개 번호 일치 확률 1/35,724
4등 4개 번호 일치 5만원 고정
5등 3개 번호 일치 5천원 고정

※ 1·2·3등 당첨금은 해당 회차 판매금의 배분 비율에 따라 당첨자 수로 나눠 지급됩니다. 당첨자가 없으면 다음 회차로 이월됩니다.

자주 묻는 질문

이 번호 추첨기는 무료인가요?
완전 무료입니다. 회원가입 없이 횟수 제한 없이 사용하실 수 있습니다.
자동 생성 번호와 직접 선택 번호 중 어느 게 더 유리한가요?
확률적으로 완전히 동일합니다. 1~45번 중 어떤 6개 조합도 당첨될 확률은 똑같이 1/8,145,060입니다. 선택 방법은 확률에 영향을 주지 않습니다.
번호를 여러 번 생성해도 되나요?
물론입니다. '번호 생성' 버튼을 누를 때마다 새로운 무작위 번호 6개가 즉시 생성됩니다. 마음에 드는 번호가 나올 때까지 자유롭게 반복하세요.
번호 생성 방식은 어떻게 작동하나요?
브라우저 내장 난수 생성기를 사용해 1~45번 중 서로 다른 6개를 무작위로 선택합니다. 외부 서버에 데이터를 전송하지 않으며, 모든 처리가 사용자 기기 내에서 완결됩니다.
로또 구매는 어디서 하나요?
전국 복권 판매점(편의점·마트 등)에서 1장 1,000원에 구매할 수 있습니다. 동행복권 공식 앱을 통해 온라인 구매도 가능합니다. 럭키툴즈는 번호 생성 도구만 제공하며 복권 판매와는 무관합니다.
